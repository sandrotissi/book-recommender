## User Interface
We built an interactive UI using Streamlit that allows users to input a User ID and see their personnal book recommendations. If a user is not yet register, we still make recommendations to them based on the Top-liked books. 
Once users are identified, they see visually appealing book recommendations complete with cover images fetched via the google book api, they also directly get informations on on the author and the synopsis of the book.
